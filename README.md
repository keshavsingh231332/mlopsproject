# student-score-prediction

This project predicts student exam performance using Machine Learning.

## Features

- Student score prediction
- Linear Regression model
- Accuracy evaluation
- User input based prediction

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn

## How to Run

Install required libraries:

```bash
pip install pandas numpy scikit-learn
```

Run the project:

```bash
python main.py
```

## Project Workflow

1. Generate student data
2. Train machine learning model
3. Evaluate model performance
4. Predict student scores

## Output

The model predicts exam scores based on:
- Study hours
- Attendance
- Sleep hours
- Previous scores
